# MathFP VS Code Extension

Provides syntax highlighting, bracket matching, and comment toggling for MathFP language files (`.mfp`) in VS Code.

## Features

- Syntax highlighting for keywords, functions, variables, numbers, operators
- Toggle line comments with `Ctrl+/`
- Bracket and parentheses auto-closing

## Installation

1. Build the extension: `vsce package`
2. Install in VS Code: `code --install-extension mathfp-0.0.1.vsix`
